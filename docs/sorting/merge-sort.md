# Merge Sort

合併排序（Merge Sort）是一種使用分治法策略的排序演算法。它的基本思想是將數列分成兩個子數列，然後對兩個子數列進行遞迴操作，直到所有的子數列都排序完成。最後，將排序完成的子數列合併成一個有序的數列。

合併排序的時間複雜度為 O(n log n)。它需要額外的記憶體空間，因此在實際應用中，合併排序的效率比快速排序差。
